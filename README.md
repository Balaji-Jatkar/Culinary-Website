# Culinary Website

A beginner-friendly culinary website that provides curated recipes by expert chefs at a reasonable price. Recipes are simple to understand and easy to follow, unlike many cooking shows.

## Features

- Curated recipe collections by expert chefs
- Affordable pricing
- Easy step-by-step recipes for beginners
- Multiple categories and food collections
- Special combinations for niche food lovers
- Quick meal options for busy parents
- Add to cart feature
- Quantity increase/decrease in cart
- Login / Logout system
- Attractive food image assets

## Built With

- HTML
- CSS
- JavaScript

## Functionalities

- Browse recipe collections
- Add items to cart
- Update item quantity
- Login and Logout
- View categories and collections

## Purpose

This website is made to help users cook easily with clear recipes, save time, and explore a wide variety of dishes in one place.
